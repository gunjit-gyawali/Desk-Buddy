# Desk Buddy

Desk Buddy is a small project built using an ESP32 and an OLED display. It's basically a tiny companion that sits on your desk and displays time, faces, animations, or anything else you program it to show.

I made this project to learn more about ESP32 programming and OLED displays while creating something fun and useful.

## Components Used

* ESP32 – Main microcontroller
* OLED Display – Displays the output
* Breadboard – Used for prototyping
* Jumper Wires – Connect all the components
* USB Cable – Powers and programs the ESP32

## How It Works

The ESP32 controls the OLED display and sends information to it. Depending on the code, the display can show time, simple animations, icons, or custom graphics.

This project is simple, but it can be expanded with many additional features.

## Wiring

OLED → ESP32

* VCC → 3.3V
* GND → GND
* SDA → GPIO 21
* SCL → GPIO 22

Note: Pin connections may vary depending on the OLED module you're using.

## Why I Built This

I wanted to build something more interesting than basic LED projects and learn how displays work with microcontrollers. The idea of having a small programmable "desk buddy" seemed like a fun project and a good way to practice working with the ESP32.

## Future Improvements

Some features I might add in the future:

* Animated faces
* Clock and date display
* Wi-Fi connectivity
* Buttons for interaction (will add)
* Different expressions and animations

## Getting Started

1. Connect the OLED display to the ESP32.
2. Upload the code to the ESP32.
3. Power the board using a USB cable.
4. Watch your Desk Buddy come to life.

## Project Goal

The goal of this project is to learn about microcontrollers, displays, and embedded programming while creating something fun to keep on a desk.

Feel free to modify the code and make your own version of Desk Buddy.
